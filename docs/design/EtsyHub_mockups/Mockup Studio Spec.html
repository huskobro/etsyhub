01Ürün vizyonu & senaryolar

Mockup Studio'nun varlık nedeni: Etsy satıcısı için "tasarım hazır → listing'e gidecek görseller hazır" arasındaki süreyi 30 dakikadan 2 dakikaya indirmek. Tek tasarım çoklu mockup. Tek tıkla bir set. Kategoriye göre doğru sahne. Kalite riski olan her durumda erken uyarı.

Vizyon (tek cümle): "Tasarım seç, kategoriyi seç, set'i seç — gerisini sistem yapar." Mockup Studio bir editör değildir; kategoriye-doğru-otomatik-üretim aracıdır. İnce ayar var ama opsiyonel; default akış %85 kullanıcı için tek tıkla çalışmalıdır.

Birincil personalar

PersonaYükEn çok kullanırÖncelik sinyali
POD satıcısı — Esra, t-shirt + tote30–50 listing/hafta"Etsy Optimized Set", batch render, ZIPHız + tutarlılık. 8–10 mockup 60sn'de.
Dijital satıcı — Ayşe, clipart bundle5–15 bundle/haftaCover grid, transparency, "best thumbnail" önerisiGörsel kalite + Etsy thumbnail gücü.
Wall art — Murat, poster + nursery10–20 tasarım/hafta, çoklu oranOran filtresi, oda sahneleri, set of 3Oran doğruluğu; yanlış orana bastırma riski sıfır.

5 temel senaryo

  1. Hızlı listing seti. Esra yeni t-shirt tasarımını seçer → T-Shirt kategorisi → "Etsy Optimized Set" → 12 mockup 90sn'de hazır → hero seçer → Listing Builder'a aktarır.
  2. Çoklu oran wall art. Murat tek poster tasarımını 4 oran (2:3, 3:4, 4:5, ISO) × 3 oda sahnesi olarak otomatik genişletir. 12 mockup. Safe area uyarısı çıkanları yeniden çerçeveler.
  3. Clipart digital preview. Ayşe 24 element bundle için "Clipart Digital Listing Set" — kapak grid + transparent + zoom + usage idea — 4 görsel olarak Etsy'ye gönderir.
  4. Bookmark seti. Dikey 2:7 oran, kesilme riski yüksek. Sistem fit-to-frame; tekli + set 3/5 + elde + kitap içi + flat-lay üretir.
  5. Saved preset. "My T-shirt go-to" 8'li set, yeni tasarımda tek tıkla yeniden uygulanır; oran uyumsuzluğunda sistem substitute önerir.

03Ana kullanıcı akışları

5 akış. Birincil akış en kısa olmalı: Studio'ya giriş ile ilk render arasında en fazla 4 tıklama.

FLOW 1 Generated design → Etsy optimized set (mutlu yol)
FLOW 2 — LOCAL UPLOAD → WALL ART
  1. Drag-drop upload
  2. Sistem oranı algılar (2:3 detected → "Vertical Poster" sub-cat)
  3. Template grid 2:3 filtreli
  4. 4 oda sahnesi seçilir
  5. Render → Output gallery → ZIP

Karar: oran tespiti otomatik; manuel "Adjust" altına gizlenir.

FLOW 3 — CLIPART BUNDLE PREVIEW
  1. 5–50 PNG yüklenir veya bookmark'tan
  2. Studio "Bundle modu"na geçer (başlık değişir)
  3. Preset: Clipart Digital Listing Set
  4. Kapak grid + transparent + zoom + usage idea

Karar: Bundle modu ikinci üst-state, UI başlığı net değişir.

FLOW 4 — BOOKMARK SETİ
  1. Tasarım seç → Bookmark
  2. Sistem dikey 2:7 kontrolü; uyumsuzsa uyarı
  3. Preset: Bookmark Launch Set (6 sahne)
  4. Safe area indicator her render'da

Karar: Bookmark için kırpma yasak; fit-to-frame zorunlu.

FLOW 5 — SAVED SET YENİDEN KULLANIM
  1. Yeni tasarım seçilir
  2. "Apply Saved Set" → "My T-shirt go-to"
  3. Oran uyumsuzluğunda otomatik substitute önerisi
  4. Direct render

Karar: Saved set'ler kategori bağımlı; cross-cat uyarı.

Preset oluşumu — sistem mantığı

Bir preset, etiket sorguları kullanarak dinamik şekilde oluşur. Statik liste değil. Örnek:

"etsy-optimized-wall-art": {
  category: "wall-art",
  aspectRatio: "{{ design.aspectRatio }}",
  composition: [
    { sceneType: "lifestyle", mood: ["boho","modern","scandi"], pick: 4 },
    { sceneType: "set-of-3",  mood: ["scandi","minimal"],     pick: 2 },
    { sceneType: "flat-lay",                                  pick: 1 },
    { sceneType: "studio",                                    pick: 1 }
  ],
  hero: { sceneType: "lifestyle", conversionScore: ">85" }
}

Sistem bu sorguyu kullanıcının tasarımına ve oranına çevirir, tam 8 template seçer ve hero'yu işaretler. Aynı tasarım iki kez basılırsa sonuç deterministiktir; yeni rastgele set isterse "Shuffle" buton.

10Batch & otomasyon

  • Multi-design batch (V1): Selection Studio'dan 5–20 generated tasarım × aynı preset. Job sayısı = tasarım sayısı.
  • Color variant fan-out (V1): T-Shirt için 1 tasarım × 4 renk = 4 ayrı job.
  • Aspect fan-out (MVP): Wall Art'ta 1 tasarım × 4 oran = 4 job (oran-aware).
  • Recurring set (V2): Yeni "generated design" geldiğinde otomatik default preset uygula.
  • Auto-listing draft (V2): Mockup ready → "Mountain Print Set — Draft" listing'i otomatik oluştur.
İlke: Otomasyon her zaman bir review adımı bırakır. Sistem hiçbir zaman kullanıcı onayı olmadan listing yayınlamaz.

13Kapanış: 10 karar, 10 risk

10 kritik karar (özet)

  1. Editör değil, üretici: Mockup Studio default akışı tek tıklamada render başlatır; ince ayar opsiyonel.
  2. Kategori birinci sınıf yurttaş: Tüm akış ve template kataloğu kategori-merkezli.
  3. Preset = dinamik etiket sorgusu, statik liste değil — yeni template eklendiğinde preset'ler kendi günceller.
  4. Oran-aware her yerde: Tasarım oranı tüm filtreleri otomatik daraltır.
  5. Safe area kanonik kontrol: Canvas + Batch Review + Editor — üç ayrı yerde aynı sözlükle gösterilir.
  6. Background render: Studio kapatılabilir; sidebar count badge.
  7. Hata sözlüğü 5 sınıf: timeout / asset / safe-area / quota / provider-down — her biri 3 önerili aksiyon.
  8. Otomasyon her zaman review bırakır: sistem hiçbir zaman onaysız listing yayınlamaz.
  9. Etsy 10 görsel limiti Send to Listing'de canlı dayatılır; 11+ render outside-gallery işaretlenir.
  10. Saved Sets kategori bağımlı: cross-category uyumsuzluğu otomatik substitute önerisiyle çözülür.
